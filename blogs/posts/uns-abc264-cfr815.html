Some unsolved problems (ABC264, CFR815)

Recently, I've joined AtCoder Beginner Contest 264 and Codeforces Round 815 (Div. 2). Here's some unsolved or almost-solved problems I've met.

ABC264 - E

This is another graph theory problem. It's very difficult to maintain the deletion of edges, so we can consider reversing the operation order of the peoblem.

Then the problem became to addition of edges. Then we can use Union Find to solve it.

CFR815 - C

I was always thinking about which place to put a "corner", but actually we don't need to do that. We can find that as long as we have two adjacent 0s, we can put a "corner" and remove an adjacent 1. So we can first determine whether there are two adjacent 0s. If there are, the answer is the number of 1s. Otherwise, Find a place with minimum 1s, remove it, and the answer is the remaining number of 1s plus 1.
