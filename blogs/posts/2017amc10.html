Problem 18

Here's another probability problem. In the first round, Amelia has \(\displaystyle\frac{1}{3}\) probability to win, and the other \(\displaystyle\frac{2}{3}\) probability to continue. If Amelia doesn't win, then go to the second round. In the second round, Blaine has \(\displaystyle\frac{2}{5}\) probability to win. If Blaine doesn't win, then repeat the process.

In conclusion, Amrelia win with probability \[\displaystyle\frac{\displaystyle\frac{1}{3}}{\displaystyle\frac{1}{3}+\displaystyle\frac{2}{3}\times \displaystyle\frac{2}{5}}\] Assume it is \(\displaystyle\frac{p}{q}\), then \[q-p=\displaystyle\frac{1}{3}+\displaystyle\frac{2}{3}\times \displaystyle\frac{2}{5}-\displaystyle\frac{1}{3}=\displaystyle\frac{2}{3}\times \displaystyle\frac{2}{5}=\displaystyle\frac{4}{15}\]

Problem 20

In option A, there's only one cases: \(999...999\) to \(1000...000\), but because \(1274\) isn't divisible by \(9\), so it is impossible.

In option B, there's also only one cases: \(2999...999\) to \(3000...000\), but because \(1274\equiv 5\pmod 9\), so it is impossible either.

In option C, there's a few cases, one of them is: \(9299...999\) to \(9300...000\), but because \(1274\equiv 5\pmod 9\), so it is impossible.

In option D, there's a possible case: \(111...1119999\) to \(111...1120000\), because \(36-1=1274-1239\), so this is the right answer.

In option E, there's a possible case: \(111...1119\) to \(111...1120\), because \(9-1\neq1274-1265\), so this is not the right answer.
